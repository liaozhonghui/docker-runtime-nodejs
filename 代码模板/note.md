## 回溯算法

概念：回溯算法实现方案是，一步一步从选择列表中进行选择，当满足条件的时候，将找到的路径加入到结果列表中

1.  原理是决策树概念
2.  基本决策概念：
    - 路径：已经做出的选择
    - 选择列表：当前可以做的选择
    - 结束条件：到达决策树底层，无法再做选择的条件

回溯算法代码框架模板

```python
res=[];
def backtrack(路径，选择列表) :
    if 满足终止条件:
        res.push(路径)；
        return;

    for 选择 in 选择列表 :
        做选择
        backtrack(路径，选择列表);
        撤销选择
```
